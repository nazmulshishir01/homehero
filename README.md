##  Key Features

- **JWT Authentication** - Secure token-based authentication for protected endpoints
- **RESTful API** - Complete REST API with proper HTTP methods and status codes
- **Advanced Filtering** - MongoDB operators ($gte, $lte) for price range filtering
- **Rating System** - Calculate and store average ratings with user reviews
- **Database Relations** - Relationships between services, bookings, and reviews
- **CORS Configuration** - Secure cross-origin resource sharing
